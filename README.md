## ✨ What is MockPrep?

**MockPrep** is an **AI-powered interview preparation platform** that simulates real interview environments for developers.

It helps users prepare for:

* 💻 **Coding Interviews**
* 🎤 **Technical Interviews**
* 🧠 **Behavioral Interviews**
* 👔 **HR Interviews**
* 📄 **ATS Resume Screening**

Instead of practicing randomly, MockPrep provides **structured interview simulations, AI evaluation, and performance analytics** to help candidates improve faster.

---

## 🔥 Core Features

### 💻 DSA Coding Practice

Practice coding problems inside a **built-in code editor** designed for interview preparation.

Features include:

* In-browser **code execution**
* **LeetCode-style problems**
* Multiple programming language support
* Real-time coding practice environment

Perfect for preparing for **technical rounds of interviews**.

---

### 🎤 AI Mock Interviews

Practice realistic interviews powered by AI.

MockPrep simulates:

* Technical interviews
* HR interviews
* Behavioral interviews

The AI generates questions and evaluates responses to help users understand their strengths and weaknesses.

---

### 📄 ATS Resume Checker

Upload your resume and receive an **ATS score out of 100**.

MockPrep analyzes resumes based on:

* Keyword relevance
* Resume formatting
* Section optimization
* Skill matching

Users receive **detailed feedback reports** to improve resume visibility for recruiters.

---

### 📊 Performance Analytics Dashboard

Track your preparation progress through a powerful dashboard.

The dashboard provides:

* Interview history
* ATS score tracking
* Performance breakdown
* Improvement insights

This allows users to **monitor growth and improve weak areas**.

---

### 🧾 Detailed Interview Reports

After each interview session, MockPrep generates a **complete performance report** including:

* Questions asked
* Answer evaluation
* Improvement suggestions
* Overall performance score

Users can revisit reports anytime to track progress.

---

### ⚡ Smart Credit System

MockPrep uses a **credit-based system** to manage AI-powered features.

Credits are used for:

* AI interview sessions
* Resume ATS analysis
* AI evaluations

This ensures efficient and scalable AI usage across the platform.
